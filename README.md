# BMW-Sales-Data-Analysis
This project analyzes BMW sales data to identify trends in revenue, model performance, and regional demand. It involves data cleaning, preprocessing, and visualization using Python and Excel tools. Interactive dashboards and reports were built to support data-driven decision-making.
# **Project Overview**

This project focuses on analyzing BMW sales data using Power BI to understand sales performance, customer preferences, and market trends. The dashboard provides interactive visualizations to support data-driven business decisions.

# Dashboard Features

Total Revenue: Displays total sales value (4B USD).

Total Sales Volume: Shows total units sold (253M).

Region Filter: Enables filtering by geographical regions.

Sales by Region: Visual representation of regional sales distribution.

Sales by Color: Analysis of car color preferences.

Yearly Sales Trend: Bar and line charts showing performance over time.

Sales Volume by Year: Tracks growth in units sold.

Price by Engine Size: Compares pricing based on engine capacity.

Price by Fuel Type: Shows price variations across fuel categories.

# Key Insights

Sales are well distributed across major regions.

Black and silver are the most preferred car colors.

Sales volume shows consistent growth over the years.

Electric and hybrid vehicles have higher average prices.

Larger engine sizes are associated with higher pricing.
